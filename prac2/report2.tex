\section{条件式}
\begin{itemize}
\item 条件式1: \verb!((a >= 2 || b == 1) && c > 0)!\\
a  \textgreater= 2 は偽、b == 1 は真であり、この論理和は真となる。

次に c  \textgreater 0 は真であり、 \verb!(a >= 2 || b == 1)!も真であるから論理積は
      真となる。

すなわち、条件式1 は真である。

\item 条件式2: \verb!(a >= 2 || b == 1 && c > 0)!\\
b == 1 と c \textgreater 0 はそれぞれ真であり、この論理積は真となる。

次に a  \textgreater= 2 は真であり、\verb!b == 1 && c > 0! も真であるか
      ら論理和は真となる。

すわわち、条件式2 は真である。  

\item 条件式3: \verb+((a >= 2 || b == 1 && c > 0) != 0)+\\
条件式2 より、 \verb!(a >= 2 || b == 1 && c > 0)! は真であるから、条件式
      3 は真となる。

\end{itemize}

\section{課題6 の確認}
プログラムのソースと実行結果を以下に示す。

{
  int a=1, b=1, c=1;

  printf("1 : %d\n",((a >= 2 || b == 1) && c > 0));
  printf("2 : %d\n",(a >= 2 || b == 1 && c > 0));
  printf("3 : %d\n",((a >= 2 || b == 1 && c > 0) != 0));

}
\end{verbatim}
\end{itembox}

2 : 1
3 : 1

\normalsize

C言語では、演算子を用いて評価される式の値の偽と真は、それぞれ 0 と 非 0 で
判断されている。

すなわち出力した値が、0 以外 であれば真、0 であれば偽ということである。

このプログラムの出力結果はすべて 0 以外であるため 条件 1 から 3 はすべて真であ
ると言える。

\section{課題6 が一定条件になる範囲}
\begin{itemize}
\item 条件式1, 2 が共に真\\
b = 1, c $>$ 0 (Aは任意)\\
または a $\ge$ 2, b = 0, c = 1
\item 条件式1 が真, 2 が偽\\
なし
\item 条件式1 が偽, 2 が真\\
a $\ge$2, c $\le$0 (Bは任意)\\
\item 条件式1, 2 が共に偽\\ 
a $<$ 2, b $\ne$ 1 , c $\le$ 0\\
または a $<$ 2, b $\ne$ 1, c $>$ 0\\
または a $<$ 2, b = 1, c $\le$ 0 
\end{itemize}
