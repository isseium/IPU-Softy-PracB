\section{型で表現できる範囲}
型によって表現できる範囲を表~\ref{tab:1}に示す。
\begin{table}[h]
\label{tab:1}
\caption{型で表現できる範囲(1)}
\begin{center}
\begin{tabular}{|l|l|}
\hline
型 & 整数の範囲\\
\hline
unsigned short int & 0 〜 65535\\
signed short int & -32768 〜 32767\\
unsigned int & 0 〜 4294967295\\
signed int & -2147483648 〜 2147483647\\
\hline
\end{tabular}
\end{center}
\end{table}

求める方法を以下に示す。
\begin{itemize}
\item unsigned （無符合）の場合\\
それぞれの型のデータサイズを n ビットとすると、
\[
 0 〜 2^{n} - 1
\]
で範囲を求めることができる。

\item signed （有符合）の場合\\
それぞれの型のデータサイズを n ビットとすると、
\[
 -2^{n-1} 〜 2^{n-1} - 1
\]
で範囲を求めることができる。

\end{itemize}
